# Pawan Funde — Portfolio Website

A dark & modern personal portfolio built with HTML, CSS & Vanilla JS.

## Folder Structure
```
portfolio/
├── index.html
├── style.css
├── script.js
├── vercel.json
└── assets/
    └── resume/
        └── Pawan_Funde_Data_Analyst_Resume.pdf
```

## Deploy to Vercel
1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import your repo
3. Click Deploy — done!

## Customise
- Add your profile photo: replace the `.avatar-initials` div in index.html with an `<img>` tag
- Connect contact form: sign up at formspree.io → get your endpoint → replace the form's `action` attribute
- Add GitHub links: replace `href="#"` in each project card with your actual GitHub URLs
